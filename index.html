Deployment Steps

🚀 Recommended: Experience Cloud

Based on your use case, we recommend deploying as an Experience Cloud site. This allows public access while maintaining Salesforce security and scalability.

Quick Start Guide

  1. Create Experience Cloud Site: Go to Setup → Digital Experiences → New
  2. Choose Template: Select "Customer Service" template
  3. Add Einstein Bot: Navigate to Builder → Components → Einstein Bots
  4. Configure Permissions: Enable guest user access for public use
  5. Customize Branding: Add your logo and colors
  6. Publish: Click "Publish" and share your site URL
